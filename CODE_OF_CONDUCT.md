# Code of Conduct

## Our standard

We want Job Applier to be a respectful, practical and welcoming project.

Participants are expected to:

- communicate with respect;
- assume good intent while giving direct feedback;
- focus on solving problems, not attacking people;
- avoid harassment, discrimination or hostile behavior.

## Enforcement

Project maintainers may remove comments, issues, pull requests or contributors that violate these expectations.
